**Class Structure and Instance Methods**

A _class_ is like a blueprint for creating objects. It defines what data and actions the objects will have.

An _instance_ is an object created from that class.

An _instance method_ is a function inside a class that works with a specific object.

Example:

```python
class Dog:
    def bark(self):
        print("Woof!")

my_dog = Dog()   # create an instance
my_dog.bark()    # call the method
```

Here, `Dog` is the class, `my_dog` is an instance, and `bark` is an instance method.
