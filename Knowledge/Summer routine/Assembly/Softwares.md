# Useful Software for the Hardware Roadmap

## Hardware Identification

### CPU-Z

Purpose:

- CPU information
    
- RAM information
    
- Motherboard information
    

Website:

- [https://www.cpuid.com/softwares/cpu-z.html](https://www.cpuid.com/softwares/cpu-z.html)
    

---

### HWiNFO

Purpose:

- Detailed hardware information
    
- Temperature monitoring
    
- Fan monitoring
    
- Sensor monitoring
    

Website:

- [https://www.hwinfo.com](https://www.hwinfo.com/)
    

---

### Speccy

Purpose:

- Quick hardware overview
    
- Easy-to-read component summary
    

Website:

- [https://www.ccleaner.com/speccy](https://www.ccleaner.com/speccy)

## Recommended Installation Order

1. HWiNFO
    
2. CPU-Z
    
3. CrystalDiskInfo
    
4. CrystalDiskMark
    
5. Cinebench
    
6. OCCT
    
7. Fan Control (optional)
    

These tools are sufficient for a beginner hardware-learning setup and cover identification, monitoring, diagnostics, maintenance, and benchmarking.
