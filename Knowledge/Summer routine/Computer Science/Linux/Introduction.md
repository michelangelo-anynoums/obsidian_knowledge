### **Introduction to Linux**

Linux is a family of ==open-source==, Unix-like operating systems based on the ==Linux kernel==, first created by ==Linus Torvalds in 1991.== It has grown into one of the most popular and versatile platforms for everything from personal computing to enterprise servers. Unlike proprietary systems like Windows and macOS, Linux is open-source, meaning its source code is freely ==available== to modify, distribute, and enhance.

![[Linux_picture.webp]]

#### **Key Concepts in Linux:**

1. **Kernel:** The core part of the operating system that interacts directly with the hardware and manages resources like memory, processes, and devices.
    
2. **Shell:** A command-line interface (CLI) that allows users to interact with the operating system through text-based commands (e.g., Bash).
    
3. **Filesystem:** The structure and organization of files and directories on the system, with Linux using hierarchical file systems like ext4.
    
4. **Package Management:** The system for installing, updating, and managing software. Popular package managers include `apt` (Debian-based), `yum` (Red Hat-based), and `pacman` (Arch-based).
    

#### **Popular Linux Distributions:**

1. **Ubuntu** (1991): Known for its user-friendliness, Ubuntu is one of the most popular Linux distros for both ==beginners and experienced users.== It uses the GNOME desktop environment and supports a wide range of hardware.
    
2. **Debian** (1993): Debian is the foundation for many other distros, including Ubuntu. It’s renowned for ==stability and security, making it ideal for servers.==
    
3. **Fedora** (2003): Known for its ==cutting-edge features==, Fedora is the upstream source for Red Hat Enterprise Linux (RHEL) and focuses on incorporating the latest technologies while maintaining a high level of security.
    
4. **Arch Linux** (2002): Arch is a ==lightweight and flexible distro that follows a rolling release model.== It's popular with advanced users who prefer to customize their systems from the ground up.
    
5. **CentOS** (2004): CentOS ==is a free==, community-driven version of RHEL, ==offering stability and enterprise-grade performance== without the commercial support.
    
6. **Mint** (2006): A user-friendly ==alternative to Ubuntu==, Linux Mint offers a more familiar desktop environment for those transitioning from Windows.
