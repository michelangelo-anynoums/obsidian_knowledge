# Lock Types, Security Features, and Common Vulnerabilities

## 1. Pin Tumbler Cylinder Locks

### Examples

- Most residential door locks
    
- Euro cylinders
    
- Rim cylinders
    
- Mortise cylinders
    

### How They Work

Pins inside the lock must align at a specific height for the cylinder to rotate.

### Security Features

- Security pins (spool, serrated, mushroom)
    
- Restricted keyways
    
- Anti-drill inserts
    
- Hardened steel components
    
- Tight manufacturing tolerances
    

### Common Vulnerabilities

- Poor manufacturing tolerances
    
- Wear over time
    
- Weak key control (easy key duplication)
    
- Cheap materials
    
- Weak installation rather than lock failure
    

### Security Level

- Low to High, depending on quality
    

---

## 2. Wafer Locks

### Examples

- Filing cabinets
    
- Mailboxes
    
- Vehicle glove compartments
    
- Some older automobiles
    

### How They Work

Flat wafers align instead of pins.

### Security Features

- Simple and inexpensive
    
- Compact design
    

### Common Vulnerabilities

- Fewer locking elements
    
- Larger manufacturing tolerances
    
- Often designed for convenience rather than security
    

### Security Level

- Generally Low
    

---

## 3. Disc Detainer Locks

### Examples

- High-security padlocks
    
- Bicycle locks
    
- Some European security cylinders
    

### How They Work

Rotating discs must align to create a path for a sidebar.

### Security Features

- Resistant to many common cylinder attacks
    
- Durable construction
    
- Often contains hardened components
    

### Common Vulnerabilities

- Poor-quality copies
    
- Wear affecting precision
    
- Manufacturing shortcuts
    

### Security Level

- Medium to Very High
    

---

## 4. Lever Locks

### Examples

- Traditional safes
    
- Older commercial buildings
    
- Some British doors
    

### How They Work

Levers must be lifted to correct heights before the bolt can move.

### Security Features

- Large internal components
    
- Can support many levers
    
- Good durability
    

### Common Vulnerabilities

- Low-lever-count designs
    
- Wear creating excessive tolerances
    

### Security Level

- Medium to High
    

---

## 5. Tubular Locks

### Examples

- Vending machines
    
- Bicycle accessories
    
- Computer security devices
    

### How They Work

Pins are arranged in a circle around the keyway.

### Security Features

- Compact
    
- Unusual key format
    

### Common Vulnerabilities

- Low-cost versions often have poor tolerances
    
- Security varies dramatically by manufacturer
    

### Security Level

- Low to Medium
    

---

## 6. Dimple Locks

### Examples

- Modern European cylinders
    
- Commercial facilities
    
- High-security residential systems
    

### How They Work

The key contains drilled dimples rather than edge cuts.

### Security Features

- Multiple pin rows
    
- Sidebars
    
- Restricted key systems
    
- Complex key profiles
    

### Common Vulnerabilities

- Budget versions may lack advanced protections
    
- Installation weaknesses
    

### Security Level

- Medium to Very High
    

---

## 7. Magnetic Locks

### Examples

- High-security commercial cylinders
    
- Specialized access systems
    

### How They Work

Magnets embedded in keys interact with internal mechanisms.

### Security Features

- Difficult key duplication
    
- Additional authentication layers
    

### Common Vulnerabilities

- Cost
    
- Proprietary maintenance requirements
    

### Security Level

- High
    

---

## 8. Electronic and Smart Locks

### Examples

- RFID systems
    
- Bluetooth locks
    
- Smart home locks
    

### Security Features

- Audit logs
    
- Remote management
    
- Multi-factor authentication
    
- Temporary credentials
    

### Common Vulnerabilities

- Firmware flaws
    
- Weak passwords
    
- Poor wireless security
    
- Power failure considerations
    

### Security Level

- Highly dependent on implementation
    

---

# Security Features Worth Knowing

## Security Pins

Create misleading feedback and increase manipulation resistance.

### Types

- Spool pins
    
- Serrated pins
    
- Mushroom pins
    
- Hybrid designs
    

---

## Sidebars

Additional locking components that must align correctly.

### Found In

- Disc-detainer locks
    
- Dimple locks
    
- High-security cylinders
    

### Benefit

Provides a second layer of security beyond standard pins.

---

## Restricted Keyways

Keys can only be duplicated by authorized parties.

### Benefit

Reduces unauthorized key copying.

---

## Anti-Drill Protection

### Includes

- Hardened steel pins
    
- Carbide inserts
    
- Hardened faceplates
    

### Benefit

Improves resistance to destructive attacks.

---

## Anti-Snap Protection (Euro Cylinders)

### Includes

- Sacrificial sections
    
- Reinforced cores
    
- Hardened bridges
    

### Benefit

Addresses a common weakness in some Euro cylinder designs.

---

# Important Security Principle

The lock itself is often not the weakest component.

A high-security cylinder installed in:

- A weak door,
    
- A weak frame,
    
- A weak strike plate,
    
- Or poor access-control procedures,
    

may provide little real-world protection.

Physical security professionals frequently spend as much time evaluating the surrounding system as they do evaluating the lock itself.
